Perfume Designer Application - Phase 1

A fully offline, desktop-based perfume design application built with Python and PyQt6.

Navigation

Perfume Pyramid Builder

Implemented Features

Perfume Pyramid Builder

  • Drag-and-drop interface for assigning fragrance notes
  • Percentage controls for each note using sliders
  • Visual pyramid structure representation
  • Search/filter functionality for notes

Sample Notes Database

20 pre-loaded fragrance notes including:

  • Top Notes: Bergamot, Lavender, Lemon, Peppermint
  • Middle Notes: Jasmine, Rose, Ylang Ylang, Cinnamon
  • Base Notes: Sandalwood, Vanilla, Patchouli, Amber

Technical Features

  • Dark theme with gold/amber accents
  • Local SQLite database for fragrance notes
  • Modular code structure for future expansion
  • Responsive layout design
